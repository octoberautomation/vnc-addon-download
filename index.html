October Automation
VNC Add-On Installer

Remote access for your
PLC Gateway machines

One script to install VNC, Tailscale VPN, and auto-register your machine with the October Automation Remote UI dashboard.

Download Script How It Works
$ curl -fsSL https://download-site-sigma.vercel.app/install-vnc-addon.sh | sudo bash
🔒

Tailscale VPN

Secure mesh networking with automatic authentication. Access machines from anywhere on your tailnet.

🖥

VNC Server

x11vnc on port 5900 with password auth, configured as a systemd service for automatic startup.

🌐

Tailscale Funnel

Public HTTPS access to your PLC Gateway without port forwarding or firewall changes.

📋

Auto-Registration

Registers with the VNC Dashboard and Remote UI. Get a claim code to link the machine to your account.

Quick Start

1

Download the script

Click the download button or use curl to fetch the installer.

2

Run as root

sudo bash install-vnc-addon.sh on a machine with PLC Gateway already installed.

3

Claim your machine

Use the claim code from the script output at remoteui.octoberautomation.com/claim.
